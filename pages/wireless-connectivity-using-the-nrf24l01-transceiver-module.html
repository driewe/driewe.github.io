Using the NRF24L01+ transceiver module

In this presentation I show you how to use the cheap and reliable NRF24L01+ transceiver module. With this module you can achieve reliable point to point communications. I am showing you the basics so you can go on creating sophisticated wireless systems, including mesh sensor arrays!



You can download the sketch used in the demo from Github (see External Resources for a link).
