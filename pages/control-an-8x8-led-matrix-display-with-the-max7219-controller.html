Control an 8x8 LED Matrix Display With MAX7219 controller 
In this presentation, I will show you how to use the MAX7219 LED controller IC with a single 8x8 matrix LED display. You can combine these two components together to create large LED matrix displays, but in this presentation we'll keep it simple. In a separate lecture, we'll expand to multi-display setups.
In this Part 1 of the presentation, I'll show you how to create shapes, simple animations and text on the screen.
