Control Arduino Robot Arm with Android App

Date Sat 10 December 2016 By Slant Concepts Tags Arduino / Robotics / Android / Tutorials
Parts Used
  • LittleArm
  • Arduino Uno
  • HC-05 Arduino Bluetooth Module
  • Android Tablet

This tutorial is based on resources from this page.

The LittleArm is a customarily controlled utilizing a free desktop PC application that keeps running on Windows and Linux. In any case, this requires the LittleArm be associated with a PC to be controlled. We at long last finished the application for the LittleArm so you can control the Arduino remotely with Bluetooth.

The application permits you to control all the DOF's of the LittleArm additionally record sets of charges and play them back. The code for the Arduino does not transform from our customary code in the product bundles from here.

The LittleArm app for this project can be downloaded here.
